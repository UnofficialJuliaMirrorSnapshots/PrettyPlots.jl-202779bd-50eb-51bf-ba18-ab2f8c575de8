This is a plotting tool for [NLOptControl.jl](https://juliampc.github.io/MPCDocs.jl/latest/).

## Installation

In Julia, you can install PrettyPlots.jl by typing:
```julia
 julia>Pkg.add("PrettyPlots")
```
